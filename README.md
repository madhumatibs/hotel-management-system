# Hotel Management System

A complete Hotel Management System for managing room bookings, customer records, check-in/check-out, reservations, billing, and hotel operations efficiently. Built to simplify hotel administration with a user-friendly interface and secure data management.

---

## Features

* User Login Authentication
* Add New Reservations
* View All Reservations
* Generate Reservation Reports
* Location-Based Report Generation
* Customer & Room Management
* Billing and Hotel Operations Management
