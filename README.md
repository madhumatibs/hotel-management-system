# Hotel Management System

This is a Hotel Management System project developed using Advanced Java and MySQL.

## Features
- Room Booking
- Customer Registration
- Check-In / Check-Out
- Billing System
- Room Availability Status
- Admin Panel

## Technologies Used
- Advanced Java
- JDBC
- MySQL
- JSP
- Servlet
- HTML
- CSS

## Purpose
College submission project for Advanced Java subject.
